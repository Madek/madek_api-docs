Media-Entries Index /media-entries/

GET

Returns a paginated Media-Entry collection.

Query Prameters

collection_id

value required
UUID as a string no

The value, a UUID of a Collection (aka. Set), filters those Media-Entries which are "children" of that Collection. See also Collection in the Madek documentation.
