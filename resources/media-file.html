Media-File /media-files/{id}

GET

Returns a subset of the properties of a Media-File. This includes the following keys:

  • content_type,
  • created_at,
  • filename,
  • media_type,
  • media_entry_id, and
  • size.

See also MediaFile in the Madek documentation.

Warning: this part of the API might change in the future.
