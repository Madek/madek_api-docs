Meta-Keys /meta-keys/

GET

Returns a Meta-Key collection.

Query Parameters

vocabulary

value required
String, id of a vocabulary no

The value of this query parameter will be used to restrict the Meta-Keys to those which belong to the Vocabulary of equal id.
