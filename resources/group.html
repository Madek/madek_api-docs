Group /groups/{id}

Each Group in Madek is related to Users. A Group in Madek is primarily used to permit Users to access other resources.

The id parameter in the URL can be either the Ñ–d or the institutional_id of the group. The id is a UUID and can be safely used as is in the URL. The institutional_id is string and requires be URL encoded in the general case.

GET

Returns the properties of a Group.

DELETE

Returns the properties of a Group.

PATCH

Updates a Group.
